# Визуалы панели «Админ школы»

- **`login-hero.svg`** — декоративная иллюстрация на экране входа. Можно заменить на **`login-hero.jpg` / `.webp`** (рекомендуется 1200×750 или 16∶10) и подключить в `components/school-admin/school-admin-login-hero.tsx` через `<Image />` или `<img />`.

Фон страницы собран на CSS (градиенты + «орбы»), без тяжёлых картинок — быстрая загрузка.
