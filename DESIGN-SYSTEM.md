# Дизайн-система проекта

Документ для воспроизведения дизайна в других проектах. Использовать при вёрстке похожих интерфейсов.

**В коде:** токены в `app/globals.css` (`:root`, классы `.ds-*`, кнопки `.ui-btn--*`). Для SCSS/Nuxt — `styles/design-system.scss`.

---

## 1. Цвета

### Основные

| Название | HEX | Использование |
|----------|-----|---------------|
| Белый | `#FFFFFF` / `--ds-white` | Фон карточек, модалок, header |
| Чёрный | `#262626` / `--ds-black` | Текст, активные элементы, бордеры |

### Акцентные

| Название | HEX | Использование |
|----------|-----|---------------|
| Красный (primary) | `#FF2E1F` | CTA, loading indicator, активные состояния |
| Красный (error) | `#F93232` | Ошибки валидации, границы ошибок |

### Нейтральные

| Название | HEX | Использование |
|----------|-----|---------------|
| Светло-серый фон | `#E9E9E9` | Фон страницы (градиент) |
| Светло-серый 2 | `#F2F2F2`, `#F5F5F5` | Карточки, блоки |
| Серый карточки | `#E8E8E8`, `#E5E5E5` | Навигация, неактивные элементы |
| Серый средний | `#DDDDDD`, `#D8D8D8`, `#D5D5D5` | Кнопки type 4/7, hover |
| Серый бордер | `#C2C2C2` | Границы header, разделители |
| Серый текст | `#7D7D7D` | Вторичный текст, placeholder |
| Тёмно-серый | `#404040`, `#3B3B3B`, `#353535` | Иконки, тёмные элементы |
| Фон footer | `#262626` → `#414141` | Градиент CTA в footer |

### Полупрозрачные

- `rgba(0, 0, 0, 0.4)` — оверлей модалок  
- `rgba(0, 0, 0, 0.3)` — blur overlay меню  
- `#E9E9E94D` — кнопки на тёмном фоне  
- `#E9E9E91A` — псевдо-фон кнопок type 3/5  

---

## 2. Типографика

- **Шрифт:** Circular, fallback `sans-serif` / в проекте подключён Inter 400/500 до загрузки Circular.
- **Веса:** 400 (Book), 500 (Medium)
- **Fluid:** классы `.ds-text-h1` … `.ds-text-caption` в `globals.css`

Формула fluid (пример):  
`font-size: min(max(14px, calc(0.875rem + ((1vw - 3.9px) * 0.1307))), 16px);`

---

## 3. Отступы и размеры

- **Контейнер:** desktop `max-width: 97.5%`, mobile `91.5%`, центрирование `margin: 0 auto` — класс `.ds-container`
- **Брейкпоинты:** mobile `1024px`, note `1540px` — в Tailwind: `lg:`, кастом `note:` (см. `globals.css`)

**Отступы:** секции 48–120px, блоки 24–48px, элементы 12–24px, мелкие 8–12px, gap 16–48px.

**Header:** margin-top 74px (desktop), 16px (mobile при скролле); padding 9–15px / 16–24px; min-height 42–70px.

## 5. Кнопки (UIButton)

Классы: `.ui-btn`, модификаторы `.ui-btn--1` … `.ui-btn--7`.

| Type | Описание |
|------|----------|
| 1 | CTA: `#FF2E1F`, белый текст, radius 6px, padding 11px 16px |
| 2 | Текстовая, прозрачный фон, 16px / 500 |
| 3 | На тёмном: обводка белая, blur, hover светлый фон |
| 4 | Серая `#DDDDDD`, hover/active по спецификации |
| 5 | Как type 3 (баннеры) |
| 6 | Контурная тёмная |
| 7 | Светлая с активной обводкой primary |

---

## 6. Карточки и блоки

- **Бренд неактивна:** градиент 145deg F5→EB→F0, radius 8–12px, высота 47–70px — `.ds-card-brand`
- **Бренд активна:** фон `#262626`, двойная обводка — `.ds-card-brand.is-active`
- **Блок контента:** белый фон, radius 40px, padding 48px 40px — `.ds-block-section`
- **Модалка:** тень `0 3px 9px rgba(47,43,61,.15)`, оверлей `rgba(0,0,0,0.4)`

---

## 7. Градиенты

- Страница: `linear-gradient(270deg, #E9E9E9 100%, #F5F5F5 100%)` — `.ds-main`
- Footer CTA: `89.71deg, #262626 → #414141`
- Баннер снизу / радиальный / карусель — см. переменные `--ds-gradient-*` в `globals.css`

---

## 8. Навигация карусели

Стрелки 40×40px, круг, фон `#E5E5E5`, hover `#D8D8D8`, иконка `#3B3B3B`. Swiper: centeredSlides, loop, breakpoints 1024/768/0 → 5/4/3 слайда, spaceBetween 12–20px.

---

## 9. Анимации

`transition: all 0.3s ease` — стандарт. Keyframes: `ds-opacity-animation`, `ds-slow-appear`, `ds-slow-appear-2`.

---

## 10. Layout (ориентир)

```
.main (градиент)
├── LoadingIndicator (#FF2E1F)
├── Header fixed (margin-top 74px)
├── main (flex-grow)
└── Footer + модалки
```

В Next: оборачивать страницу в `.ds-main` при необходимости.

---

## 11. Иконки

SVG, цвет через `filter` или `fill`. Размеры 10–24px inline, 36–40px для кнопок.

---

## 12. Формы

`border: 1px solid #262626`, `border-radius: 8px`. Ошибка: `#F93232`. Placeholder: `#7D7D7D`, `#ACACAC` — класс `.ds-input`.

## Соответствие в `robomind-front`

| Требование ТЗ | Реализация |
|---------------|------------|
| Фон страницы, градиент E9→F5 | `body` + `.ds-main` → `var(--ds-gradient-page)` |
| Текст / бордеры | `text-ds-black`, `border-ds-gray-border` |
| CTA, ошибки | `ui-btn--1`, `text-ds-error`, `.ds-input--error` |
| Типографика | `.ds-text-h1` … `.ds-text-caption`, Inter 400/500 |
| Контейнер | `.ds-container` (97.5% / 91.5% ≤1024px) |
| Header ученик / админ школы | `bg-ds-white`, бордер `#C2C2C2`, padding ~9–15 / 16–24px |
| Супер-админ header | `.ds-header-dark` = градиент footer CTA (#262626→#414141) |
| Карточки секций | `.ds-block-section` (radius 40px) |
| Кнопки | `.ui-btn` + модификаторы `--1`…`--7` |

Панели **не** подменяют фон страницы сплошным серым — виден общий градиент ТЗ.
